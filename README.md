## 🙌 I am Cunliang Geng

I am a research software engineer with a background in bioinformatics and computational biology. My passion lies in software engineering and machine learning, and I love applying them to solve scientific challenges. 

Currently, I work at the [Netherlands eScience Center](https://www.esciencecenter.nl). There I code for scientific projects in the life sciences and medicine, train researchers in digital competence, promote digitalisation in academia, and collaborate with scientists at universities and research institutes to solve challenges in the field.


### 🔥 Open-source projects I am working on:
- [TaxoTagger](https://mycoai.github.io/taxotagger/latest/), a python library powered by deep learning and semantic search for scalable DNA barcode identification.
- [Vantage6](https://vantage6.ai/), a python infrastructure for federated analysis and learning.
- [NPLinker](https://nplinker.github.io/nplinker/latest/), a python framework for integrating multi-omics data to discover new microbial natural products.


### ✅ Open-source projects I have contributed to:
- [DeepRank](https://github.com/DeepRank), a deep learning framework for data mining 3D protein-protein interfaces.
- [MatchMS](https://github.com/matchms/matchms), a python library for processing and similarity evaluation of mass spectrometry data.
- [STRAP](https://fhir-layer.readthedocs.io/en/latest/), a data harmonisation microservice to transform proprietary healthcare data into the standard [FHIR](https://hl7.org/fhir/) format to facilitate self tracking for prevention and diagnosis of heart disease.
